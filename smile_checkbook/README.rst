Checkbook Management
====================

Generate and follow checkbooks


Usage
=====

You can create checkbook by:

* specify a range of numbers
* specify a quantity and a start number

Manually follow your checks by updating status:

* Available
* Used
* Lost
* Destroyed
* Stolen
